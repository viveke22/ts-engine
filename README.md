# TS Engine

Blazing-fast native TypeScript runtime written in Go — no Node.js, no JavaScript VM.

### Current Features
- Variables & assignment
- `console.log`
- Arithmetic operations (+ − × ÷ %)

### Roadmap
- Functions, loops, objects
- Native single-file binaries
- AI-powered compiler optimizations
- Open-source community growth

### Quick Start
```bash
go install github.com/viveke22/ts-engine@latest
ts-engine run examples/hello.ts
